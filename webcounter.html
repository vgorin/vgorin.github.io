Overview

This is a server-side application, hosted on kiev-app.us.to, which consists of an image http://kiev-app.us.to/WebCounter/CounterServlet?accountId=<accountId>&pageId=<pageId>[&showCounter] and statistics page http://kiev-app.us.to/WebCounter/CounterServlet?accountId=<accountId>&showStatistics

How To Use

Put an image provided above into your web-page, use any combination of characters for accountId and pageId; use showCounter flag to display counter digits in the image. Use accountId you've chosen earlier to access statistics page provided above.

Example

Image src: http://kiev-app.us.to/WebCounter/CounterServlet?accountId=vgrn-at-i-com-ua&pageId=webcounter&showCounter

Statistics page URL: http://kiev-app.us.to/WebCounter/CounterServlet?accountId=vgrn-at-i-com-ua&showStatistics
